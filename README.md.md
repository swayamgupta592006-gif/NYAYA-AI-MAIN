## 📌 What Is NYAYA AI?

**NYAYA AI** is a full-stack AI-powered legal assistant that makes legal knowledge accessible to every citizen. Whether you need to understand your constitutional rights, generate a legal document, or predict the outcome of a case — NYAYA AI has you covered.

---

## ✨ Features

### 🤖 AI Legal Chatbot
- Ask any legal question in plain language
- Get accurate, reliable answers instantly
- Supports **multiple languages** — accessible to everyone

### 📚 Know Your Constitutional Rights
- Complete guide to Fundamental Rights
- Easy to understand explanations
- Searchable database of Indian Constitution articles

### 📄 Legal Document Generator
- Generate FIR drafts instantly
- Bail application generator
- Legal notice templates
- Professional formatting — ready to submit

### ⚖️ Case Outcome Predictor
- AI-powered prediction based on case details
- Helps citizens understand their legal standing
- Data-driven insights

### 🌍 Multi-Language Support
- Available in multiple Indian languages
- Breaking the language barrier in legal access

## 🚀 Getting Started

### Prerequisites
```bash
node >= 18.0.0
pnpm >= 8.0.0
```

### Installation
```bash
# Clone the repository
git clone https://github.com/swayamgupta592006-gif/NYAYA-AI-MAIN.git

# Navigate to project
cd NYAYA-AI-MAIN

# Install dependencies
pnpm install

# Start development server
pnpm dev
```
